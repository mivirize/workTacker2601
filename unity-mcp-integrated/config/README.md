# 設定管理

## ファイル構成

- `mcp-servers.json`: MCPサーバーの統一設定
- `environment.json`: 環境変数のテンプレート

## mcp-servers.json

すべてのMCPサーバーの設定を一元管理するファイルです。

### 構造

```json
{
  "servers": {
    "server-name": {
      "enabled": true,
      "type": "stdio|sse",
      "command": "実行コマンド",
      "args": ["引数リスト"],
      "env": {
        "環境変数": "値"
      },
      "config": {
        "サーバー固有設定": "値"
      }
    }
  }
}
```

## environment.json

環境変数のテンプレートファイルです。実際の値は.envファイルまたはシステム環境変数で設定してください。

### セキュリティ注意事項

- APIキーや秘密情報は直接設定ファイルに記述しない
- 環境変数またはセキュアな設定管理システムを使用
- 設定ファイルはバージョン管理にコミットしない（テンプレートのみ）
